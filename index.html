PermoCombo

Calculate permutations and combinations at lightning speed with this calculator (which uses dynamic programming to speed up calculations).

A permutation is a selection of r items from a set of n items where the order in which the items are selected matters.

A combination is a selection of r items from a set of n items where the order in which the items are selected does not matter.
